DAPLOYER

Daployer is a two part project for ADOM Inc.

Part I

⊳ Overview

  • It is a desktop application IDE to facilitate the creation of Smart Contracts

    and decentralized applications within the MAIA ecosystem.

  • It leverages the ethereum blockchain.

⊳ Implementation Details

  • Daployer is developed in Java using the Java Swing GUI widget toolkit.
  • It uses the Web3 Java Ðapp API to interact with the Blockchain.

⊳ Features & Applications

  • User-friendly interface to deploy a new Smart Contract into the Blockchain.
  • User-friendly interface to interact with an existing Smart Contract and its functions.
